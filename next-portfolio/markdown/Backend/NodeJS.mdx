# NodeJS

## Create a Server
- Enter `npm init -y`, it will create a JSON script file:
```JSON
{
  "name": "cpat",
  "version": "1.0.0",
  "main": "index.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "keywords": [],
  "author": "",
  "license": "ISC",
  "description": ""
}
```
- Install node ecosystem `npm install express`
- Create server.js
```JS
// server.js
const express = require('express');
const app = express();
const PORT = 8383;

app.listen(PORT, () => {
    console.log(`Server has started on: ${PORT}`);
});
```
- Boot the server `node server.js`.

### Developer Dependency
- Enter `npm install --save-dev nodemon`. It will restart the server after you have made any modifications.
