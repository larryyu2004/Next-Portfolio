# Lab1 SystemCall
-  System call is not normal function, it will jump into the kernel
---
#### fd = open("out", 1)
1. fd -> file descriptor
2. open() -> a system call to open file, open filename = "out"
3. 1 -> read mode
---
#### write(fd, "hello/n", 6)
1. write() -> system call to write file
2. fd -> write in fd
3. "hello/n" -> write hello/n
4. 6 -> apply for 6 bytes address for writing
---
#### pid = fork();
1. pid -> process identifier
2. fork() -> returns the process identifier (pid) of the new process
3. pid > 0 -> parent process
4. pid = 0 -> child process
