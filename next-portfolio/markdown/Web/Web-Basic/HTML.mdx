# HTML

## HTML Form

### Get

```HTML
<!-- save as get-form.html -->
<!DOCTYPE html>
<html>
<head>
  <title>GET Form Example</title>
</head>
<body>
  <h2>GET Form</h2>
  <form action="/submit" method="get">
    First name: <input type="text" name="firstname"><br>
    Last name: <input type="text" name="lastname"><br>
    <input type="submit" value="Submit">
  </form>
</body>
</html>
```

- When the user submits this form, the browser will make a request to: `/submit?firstname=John&lastname=Doe`
- This data is visible in the URL, making it useful for bookmarking or debugging.
- Can be viewed and modified by user
- Has length restrictions
- Remains in the browser history

### Post
```HTML
<!-- save as post-form.html -->
<!DOCTYPE html>
<html>
<head>
  <title>POST Form Example</title>
</head>
<body>
  <h2>POST Form</h2>
  <form action="/submit" method="post">
    First name: <input type="text" name="firstname"><br>
    Last name: <input type="text" name="lastname"><br>
    <input type="submit" value="Submit">
  </form>
</body>
</html>
```
- When the user submits this form, the browser sends the data in the body of the HTTP request, not in the URL.
- Hidden from user
- No length restrictions
- Not retained in browser history
- The request looks like this (in simplified form):
```Text
POST /submit HTTP/1.1
Content-Type: application/x-www-form-urlencoded

firstname=John&lastname=Doe
```
