## Component Script

### Lifecycle Method

- **Awake()**: Called when the script instance is first loaded. (Earliest callback)
- **OnEnable()**: Called when the object or component becomes enabled.
- **Start()**: Called before the first frame update, after OnEnable().
- **FixedUpdate()**: Called at a fixed time interval, independent of frame rate.
- **Update()**: Called once per frame.
- **LateUpdate()**: Called once per frame, after all Update() calls.
- **OnDisable()**: Called when the object or component becomes disabled.
- **OnDestroy()**: Called when the object is destroyed.

```CSharp
using UnityEngine;

public class Move : MonoBehaviour
{
    private void Awake()
    {
        Debug.Log("awake");
    }

    private void OnEnable()
    {
        Debug.Log("OnEnable");
    }
    // Start is called once before the first execution of Update after the MonoBehaviour is created
    void Start()
    {
        Debug.Log("Start");
    }

    // Update is called once per frame
    void Update()
    {
        // Debug.Log("Update");
    }

    private void LateUpdate()
    {
        // Debug.Log("LateUpdate");
    }

    // 0.02 second
    private void FixedUpdate()
    {
        // Debug.Log("FixedUpdate");
    }

    private void OnDisable()
    {
        Debug.Log("OnDisable");
    }

    private void OnDestroy()
    {
        Debug.Log("OnDestory");
    }
}

```
